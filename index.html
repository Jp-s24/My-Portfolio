Content Based Image Retrieval

This project involved collaborating with team members to read barcodes from a dataset of 100 images and implementing Radon Barcode Transformation to efficiently sort and categorize them in just 25 seconds. Additionally, I used threshold projections and hamming distance algorithms to compare barcodes, achieving an 88 percent accuracy rate in retrieving images. I also optimized the search algorithm, achieving a time complexity of O(n).

Online Book Store

I developed an Online Book Store using Flask, Python, and SQLite3, creating a user-friendly web application for managing and filtering books. Users can easily browse, search, and filter books using various criteria. I utilized version control with Git and GitHub, collaborated on branching strategies, and ensured bug fixes within the development team. The project also incorporated an SQL database to store inventory, account information, and transaction data locally.
